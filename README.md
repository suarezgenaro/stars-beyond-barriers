# Stars Beyond Barriers

Website for the *Stars Beyond Barriers* astronomy outreach and mentorship initiative.

## About

Stars Beyond Barriers aims to inspire and support students from underrepresented groups, particularly Latina women, to pursue STEM through mentorship, community engagement, and connections to established science programs and research opportunities.
